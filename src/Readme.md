## Source Code for Line Follower Bot

The source code titled **Pathfinder.ino** is based on the QTR sensor array library and the TB6612FNG motor driver, with a PID control system integrated into the code. This version is optimized for competitions.

The custom programs do not use any external libraries, resulting in lower memory consumption compared to the previous version. The **custom weights** method is used to calculate the position of the bot from sensor values.

The **custom_pathfinder** program implements a PID-controlled system built entirely from scratch. These programs are still under development and are not yet fully optimized.
